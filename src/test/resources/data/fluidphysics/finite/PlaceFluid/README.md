# 测试：流体物理-有限模式-原版-放置流体

## 测试描述

该测试用于验证流体放置算法的正确性。

该测试模拟了一个玩家将指定水/岩浆方块放置到世界上时，算法是否能够正确寻找到可填充流体的空间并判断是否能够放置。

## 输入格式

第一行包含三个整数 $H$、$R$、$C$，分别表示层数、每层行数和每层列数。

第二行包含三个整数 $X$、$Y$、$Z$，表示需要放置流体的位置，
对应文本中第 $Y+1$ 层、$Z+1$ 行、$X+1$ 列的位置。

第三行包含一个字符串，表示放置的方块，不能处于垂直下落状态，必须是水或岩浆。

接下来 $H \times (R+1) -1$ 行包含 $H$ 个之间用**单个空行**间隔的 ，
内部固定为连续 $R$ 行，每行共 $C$ 个方块的用字符序列表示的平面。
每个平面所属的行的末尾可以附加上 **#内容** 以进行注释。

再之后的行不会被读取，可以填写任意内容。

## 答案格式

答案仅包含两种输出：`Y` 或 `N`

技巧：答案的首个输入必须是 `Y` 或 `N`，其余空白地方可以填写任意东西，而不会被读取。

## 说明/提示

#### 样例 1 解释：

Y = 1 层均为单层水，拥有充足的空间放置**㵘**，也就是满格水（Quanta = 8, Level = 0）方块。

---

#### 约定

对于世界外围的方块，默认均为黑曜石。

要求 $0 \leq H,R,C \leq 2147483647$，$0 \leq X \le C$，$0 \leq Y \le H$，$0 \leq Z \le R$ 。
