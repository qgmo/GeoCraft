# 测试：工具类-区块-邻居光照

## 测试描述

该测试用于验证 `ChunkUtil.getNeighborsLightFor` 方法的正确性。

该方法计算指定位置的六个面相邻方块（上、下、东、西、南、北）的光照值，并返回其中的最大值。支持方块光照（block light）和天空光照（sky light）两种类型。

## 样例格式

输入格式为一个描述光照分布的格文件（`.in.格`），使用亮度构造器的词块系统表示光照等级。可用词块及其对应的光照等级如下：

| 词块 | 光照等级 |
|------|----------|
| `零` | 0 |
| `一`~`九` | 1~9 |
| `十` / `A` | 10 |
| `B`~`F` | 11~15 |

在格文件的元数据扩展（ext）处，需填写以下字段：

- `at`：一个三元整数组，表示要查询邻居光照的位置坐标。
- `type`：字符串，光照类型。可填 `block`（或 `方块`）表示方块光照，`sky`（或 `天空`）表示天空光照。
- `expected`：整数（0-15），期望的最大邻居光照值。

该测试为输入输出合一格式，期望结果通过 `ext.expected` 字段给出，无需单独的答案文件。

## 样例 #1

### 输入 #1

```
---
name: 测试基本的方块光照
ver: 1
ext:
  at: [2,2,2]
  type: block
  expected: 15
---
F F F
F F F
F F F

F F F
F 零 F
F F F

F F F
F F F
F F F
```

## 样例 #2

### 输入 #2

```
---
name: 嵌入天花板的方块光照
ver: 1
default: 零
ext:
  at: [2,2,2]
  type: block
  expected: 14
---
E E E
E E E
E E E

略2
```

## 说明/提示

### 样例 1 解释：

3×3×3 的光照网格，中心位置 `[2,2,2]` 的光照为 `零`（0），其六个面相邻方块的光照均为 `F`（15）。因此最大邻居光照为 15。

### 样例 2 解释：

用 `default: 零` 将默认光照设为 0。仅底层（第一层，文件中最先出现，Y 最小）所有位置光照为 `E`（14），上面两层通过 `略2` 跳过，填充为默认值 `零`（0）。查询位置 `[2,2,2]` 的下方邻居（第一层）光照为 14，其余五个邻居均为 0，因此最大邻居光照为 14。
